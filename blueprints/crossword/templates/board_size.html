Crossword:

This program was written with the intention of finding if a given crossword puzzle has a solution or not. At first the user is prompted to insert the size of the board in question, then after that the user has to give the layout of the board, and lastly the user has to input the number of words and the words that will be tested.

1- In this first step, you will provide the size of board, indicading the number of rows and the number of columns.
